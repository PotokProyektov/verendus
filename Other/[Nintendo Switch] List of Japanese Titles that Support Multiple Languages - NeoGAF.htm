Originally Posted by JStein0222

I was having trouble earlier figuring out how to make a Japanese NNID, anybody know the best way to go about that? Also the easiest way to add eshop credit to an alt region would probably be good info to have

Very easy. Go to accounts.nintendo.com. Create a new account with an email you didn't use before. Set the region to japan. Log into the account on your switch (make sure you have set up a new user for this). Set the region of your switch to japan. Log into the eshop and it works. The eshop will complain that the current language setting (english or whatever yours is) isn't supported and it will show the one it does (japanese). Once you are finished you can set the region of the switch back to the one you used before. The region of the switch is what decides what eshop is loaded.
